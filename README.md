Introdution:
 ROS is short for Robotic Operation System,
 ROS is not a real operating system since it goes on top of Linux Ubuntu.

-------------------------------------------------------------------------

How to install ROS:

-------------------------------------------------------------------------
#1 Download Virtual Box:

 Download the upto date version platform packages,
 6.1.36 is the latest as of now on this site:

 https://www.virtualbox.org/wiki/Downloads

#2 Download Ubuntu 18.04 desktop image:

 on this site:

 https://releases.ubuntu.com/18.04/

#3 Create the virtual machine:

 Check for the systems requirements on this site:

 https://help.ubuntu.com/community/Installation/SystemRequirements
 https://ubuntu-mate.org/about/requirements/

 Then,
  1- go to Oracle Virtual Machine Box and click on new
  2- then write the name that will best describe your new VM
  3- select the Machine's destination folder
  4- select the type of operating system that you will install and it's version

#4 Install ROS:

 From this website download Neotic or Melodic, I went for Neotic,
 and select platform Ubuntu:

 http://wiki.ros.org/ROS/Installation

 then follow the instructions on the site:

 http://wiki.ros.org/noetic/Installation/Ubuntu

#5 Create a workspace for catkin:

 Follow the instructions on this site:
 
 http://wiki.ros.org/catkin/Tutorials/create_a_workspace
